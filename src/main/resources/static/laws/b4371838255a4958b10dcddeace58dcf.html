生活垃圾分类管理规范

  1. 范围
    本标准规定了生活垃圾分类的基本要求、分类设施设置、收运、处置及管理要求。
    本标准适用于实施生活垃圾分类管理区域。
    本标准不适用于生活垃圾的集中处置。

  3. 术语和定义
    下列术语和定义适用于本标准。
    1. 生活垃圾分类管理区域
      由市和区、县（市）人民政府按照稳步推进原则，向社会公布的实施生活垃圾分类的具体区域。
    2. 集中处置
      将生活垃圾统一运至专业的集中处置场所进行无害化处理。
    3. 生活垃圾
      人类在生活活动过程中产生的垃圾。
    4. 可回收物
      未污染的适宜回收和资源利用的生活垃圾。
    5. 有害垃圾
      对人体健康或者自然环境造成直接或者潜在危害的生活垃圾。
    6. 餐厨垃圾
      餐饮经营者、单位食堂等生产过程中产生的餐厨废弃物，居民家庭生活中产生的厨余垃圾和集贸市场产生的易腐性垃圾等有机垃圾。
    7. 其他垃圾
      除可回收物、有害垃圾和餐厨垃圾之外的其他生活垃圾。
    8. 垃圾收集容器
      符合垃圾分类和垃圾清运要求，专门用于收集生活垃圾的容器，包括分类垃圾桶（箱）、废物箱及餐厨废弃物收集专用桶等。
    9. 垃圾投放点
      按规定放置分类垃圾桶，直接投放生活垃圾的地点。
    10. 垃圾容器间
      临时集中存放生活垃圾收集容器的构筑物，便于集中收运垃圾。
    11. 垃圾收运集置点
      专门用于垃圾清运桶车对接的临时场地。
    12. 大件垃圾
      体积较大、整体性强，需要拆分再处理的废弃物。
    13. 装修垃圾
      居民装修、装饰房屋过程中所产生的废弃物。
    14. 园林绿化垃圾
      各种绿化场所进行园艺修剪或季节变化产生的落叶、树枝等。
  4. 分类类别
    1. 生活垃圾一般分为可回收物、有害垃圾、餐厨垃圾、其他垃圾四类。
    2. 特殊垃圾为大件垃圾、装修垃圾、园林绿化垃圾。

      4. 标识应准确并保持清晰和完整。
  6. 分类设施配置
    1. 居住区
      居住区内应按附录B设置四类分类收集容器。鼓励小区采用智能化垃圾分类设施。
      1. 垃圾投放点

        6.1.1.2餐厨垃圾、其他垃圾收集容器应采用240L分类垃圾桶；可回收物和有害垃圾收集容器可根据居住区实际情况采用箱、桶等多种形式。
        6.1.1.3垃圾投放点应设置在不影响居民生活和出行的位置，宜设在道路隐蔽处或单元出入口，位置应固定，不可随意移动。
        6.1.1.4垃圾投放点地面应硬化，分类垃圾收集容器摆放统一，标识朝外，便于居民投放垃圾。
      2. 垃圾容器间
        6.1.2.1 新建居住小区应按每100户不少于4㎡的面积标准配置垃圾容器间，且单个垃圾容器间面积不小于18㎡。
        6.1.2.2 垃圾容器间设置位置应便于机械化作业、垃圾分拣，应设有完善的给排水、通风和除臭消毒设施，并具有清洗功能。
      3. 特殊垃圾临时堆放点
        6.1.3.1 居住小区应设置1处固定的特殊垃圾临时堆放点。
        6.1.3.2 临时堆放点应相对隐蔽、有围档，围挡高度不得超过2.5m。
        6.1.3.3 临时堆放点应设有告示牌，公布清运服务及责任人电话、清运频次、清运时间等信息。
        6.1.3.4 临时堆放点堆放垃圾不得超过围护高度，应及时清运，确保安全。
      4. 垃圾收运集置点
        6.1.4.1封闭的居住小区内，应设置垃圾收运集置点。开放式居住小区垃圾日产量超过2000kg应单独设置垃圾收运集置点；垃圾日产量小于2000kg的，可与相邻小区联合设置垃圾收运集置点。
        6.1.4.2生活垃圾收运集置点应便于垃圾清运车辆作业和进出，做到因地制宜、布局合理、减少对交通、环境及周边居民的干扰。
        6.1.4.3集置点应设置明显标识标线，面积不小于20m2，可容纳不少于10个240L分类垃圾桶。
        6.1.4.4集置点应配有固定垃圾清运车辆停车位，满足垃圾清运作业空间需求。
        6.1.4.5集置点应设有公示牌，公布清运服务及责任人电话、清运频次、清运时间等信息。
    2. 办公区
      1. 政府机关、学校、企事业单位办公区域应设置四类垃圾收集容器；办公楼走廊和公共区域宜设置可回收物和其他垃圾两分类收集容器。
      2. 政府机关、学校、企事业单位应配置垃圾容器间，垃圾容器间规模应符合本单位实际使用需求，不得将本单位垃圾桶堆放至道路两侧。
      3. 单位食堂、餐饮企业应设置餐厨废弃物收集桶。
      4. 餐厨废弃物收集桶按附录B要求设置，使用硬质且不易腐蚀的材料，具有较好的气密性。
    3. 公共场所
      1. 机场、码头、车站、商场、市场等公共场所应设置四类垃圾收集容器。应根据人流量和垃圾产生量确定收集容器容量、配置距离。
      2. 农贸市场、果蔬批发市场应配置餐厨垃圾就地处理或预处理装置，处理场地应设有完善的给排水、通风和除臭设施，做到安全运营。
      3. 道路、街巷、河道游步道、广场应按功能和人流量大小设置统一的分类废物箱。分类废物箱须设有可回收物、其他垃圾两类。
      4. 按道路功能和人流量大小设置分类废物箱，其中商业、金融业街道宜按每100m设置一个分类废物箱；主干道、次干道、有辅道的快速路宜按每200m设置一个分类废物箱；支路、有人行道的快速路宜按每400m设置一个分类废物箱。
  7. 分类投放方法
    宜按附录C要求进行投放。
  8. 垃圾收运要求
    1. 可回收物和有害垃圾应当定期定点收集，餐厨垃圾和其他垃圾应当每天定时收集，做到日产日清。
    2. 在收集垃圾时，应做到密闭、分类收集，防止二次污染。
    3. 垃圾收集后，应及时清理作业现场、清洗垃圾收集容器，保证环境整洁。
    4. 垃圾产生单位应保证垃圾运输通道通畅，确保垃圾进行正常收运。
    5. 车辆要求
      1. 作业车辆应当具备分类收贮生活垃圾的功能，或者按照分类后不同的类别，分别配置相应的作业车辆。
      2. 作业车辆应当标识明显的分类收集、运输标识，并保持功能完好、外观整洁。
    6. 作业要求
      1. 在收集垃圾时，应尽量避开清晨、中午时间，并减少噪声。
      2. 收集生活垃圾后，将生活垃圾收集容器复位，清扫作业场地。
      3. 垃圾运输车应加装污水收集装置，收集生活垃圾后，至指定地点进行压缩排水。
      4. 运输垃圾应尽量避开上下班高峰期，使用密闭的运输工具，不得在收集、运输过程中进行敞开式压缩、分拣、转运。
      5. 在运输过程中无垃圾扬、散、拖、挂和污水滴漏，并按规定将垃圾运送至有关部门确定的处置场所。
      6. 道路沿线垃圾收运集置点上，垃圾桶摆放时间不得超过垃圾清运时间前、后30分钟。
  9. 管理要求
    1. 达标小区
      居住小区整体环境整洁，垃圾分类设施规范，垃圾分类居民参与率较高。
    2. 示范小区
      居住小区整体环境较好，垃圾分类设施规范，无乱丢垃圾、垃圾满溢等现象，垃圾分类质量较好，垃圾分类减量的经验与做法具有推广价值，能起到示范引领作用。具体指标见附录D。
    3. 办公区

办公区整体环境整洁，垃圾分类设施规范，单位职员垃圾分类参与率较高，单位食堂餐厨废弃物应根据属地环卫部门指定的收运单位进行运输。

  1. （参考性附录）
    生活垃圾分类投放方法

可回收物投放方法

  1. 纸张类：硬纸折好压平、报纸杂志等用绳索捆牢收集。
  2. 塑料类：塑料瓶瓶盖要拧开、瓶内残留物用水洗净，瓶身压扁；塑料桶、塑料盒等洗净晒干。
  3. 玻璃类：玻璃瓶要去掉瓶盖、撕掉瓶身标签，瓶内残留物用水洗净；破玻璃片用纸张包住并用胶纸缠好；玻璃应装入透明塑料袋。
  4. 织物类：废旧纺织品洗净，折好压平后用绳索捆牢。
  5. 金属类：易拉罐、罐头盒等踩扁压实，金属尖利器物用硬纸包裹捆绑，软管类进行简单剪裁。

有害垃圾投放方法

  1. 废电池：废电池保持完整，防止有害物质外漏，采用塑料袋封装，防止风吹、雨淋、暴晒。
  2. 废灯管:废灯管保持完整、清洁、干燥，用硬纸包好，防止灯管破碎。
  3. 过期药品：软膏类药品将药膏从容器中挤出，收集在信封内封好；玻璃容器类药品撕掉完整标签，分门别类装入不透明的塑料袋内；针剂、水剂类注射药品不开启，连同其完整外包装一起，封入密闭的纸筒内，针头另外放入密闭纸筒内；喷雾剂类药品在户外空气流通较好的地方，避免接触明火的条件下彻底排空；其他类药品撕掉药品标签及相关个人标签，用不透明塑料袋装好。
  4. 杀虫剂罐：杀虫剂要用完或排空，在通风良好避免明火的地方在罐身上打孔后投放。

餐厨垃圾投放方法

餐厨垃圾滤水袋装后投放到指定收集容器，避免混入包装袋、餐巾纸、保鲜膜等物品，投放时袋口要扎紧。

其他垃圾投放方法

将其他垃圾单独区分，避免将可回收物、有害垃圾混入，袋装后投放到指定收集容器内，投放时袋口要扎紧。

  1. .小区垃圾桶（房）等垃圾分类设施均为集中分类投放点，按餐厨垃圾（绿）桶、其他垃圾（黄）桶、可回收物（蓝）桶和有害垃圾（红）桶配置齐全，干净整洁。
  2. .可回收物按废旧衣物、玻璃、金属、纸类、塑料等细化类别，实行统收统运，设有专门的告示牌公布低价值物回收种类、服务电话等信息。
  3. .规范设置至少1处居民装修（大件).垃圾投放点，堆放垃圾不得高于围护高度，及时清运，确保安全。

宣传指导
  1. .小区各出入口应设有垃圾分类宣传栏，小区显著位置、各楼道、投放点及时公布更新生活垃圾分类信息，设置到位、信息公开，宣传内容完整、无破损，配有电子显示屏的小区应不定期播放垃圾减量分类宣传片或宣传标语。
  2. .应发放垃圾分类宣传和指导手册，入户率达100%。
  3. .应设有环保宣传教育基地和专职宣教员。
  4. .每周应开展公益、志愿活动制度；
  5. .应组建生活垃圾分类信息互动平台。
  6. .应建立垃圾分类楼道长、物业管理员、志愿者、督导员每周现场指导和监督工作机制。
  7. .小区宣传培训覆盖率达到100%，居民知晓率达到90%。

监督管理
  1. .应建立自查自纠工作机制，每月评价并公开小区垃圾分类工作，对监督中发现的问题、市民投诉、媒体曝光等问题，及时调查、核实、处理，督促问题责任人落实整改。
  2. .应及时制止垃圾“收运延时、运输混装、垃圾落地、沿途撒漏、作业扰民”等问题。

  4. .应建立垃圾分类诚信体系，设有专职员协助取证和执法。
